외 않돼?

JS 기반의 프로젝트를 수행하였습니다.

    Chocoham 블로그

  • SEO 최적화를 위해 sitemap 및 RSS를 작성하였습니다. sitemap은 동적으로 접근 가능하며 RSS는 xml과 json 2가지 형태로 제공됩니다.
  • 별도의 CDN 서버를 두어 이미지 로딩시간 및 웹서버의 부하를 줄였으며 계층형 트리구조를 활용해 댓글-대댓글 로직을 직접 구현하였습니다.
  • stateless 세션과 middleware를 이용해 로그인을 구현하여 관리자 페이지에서 포스트의 작성, 수정, 삭제가 모두 가능하도록 하였습니다.
  • 관리자 계정정보 및 각 댓글의 비밀번호와 같은 민감한 정보는 암호화되어 DB에 저장됩니다. 암호화 알고리즘으로는 bcrypt를 채택하였습니다.

  • vanila JS, CSS만을 이용한 싱글페이지 애플리케이션입니다. Three.js를 제외한 모든 애니메이션 및 기능을 라이브러리 없이 구현하였습니다.
  • Three.js를 이용해 웹 환경에서 3D 모델을 출력하고 브라우저 이벤트에 모델의 애니메이션을 연계하였습니다.

  • intersection observer 기능을 활용한 TOC(table of contents)를 두어 각 페이지의 주제를 나타내고 페이지간 이동을 용이하게끔 하였습니다.
