스크롤을 천천히 내려주세요.

  • SEO 최적화를 위해 sitemap 및 RSS를 작성하였습니다. sitemap은 동적으로 접근 가능하며 RSS는 xml과 json 2가지 형태로 제공됩니다.
  • stateless 세션을 이용해 관리자 로그인을 구현, 관리자 페이지 및 관련 api 요청에 제한을 두었습니다.
  • 이미지 출력을 위한 별도의 CDN 서버를 두어 이미지 로딩시간 및 웹서버의 부하를 줄였습니다.
  • 계층형 트리구조를 활용해 댓글-대댓글 로직을 구현하였습니다.
