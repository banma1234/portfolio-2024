    "Chocoham 개발 블로그"

  • Next.js, mongoDB를 이용한 CRUD 애플리케이션입니다. 각 페이지의 특성과 데이터 접근권한에 따라 SSG, SSR, CSR을 적재적소에 활용하였습니다.
  • SEO 최적화를 위해 sitemap 및 RSS를 작성하였습니다. sitemap은 동적으로 접근 가능하며 RSS는 xml과 json 2가지 형태로 제공됩니다.
  • stateless 세션을 이용해 관리자 로그인을 구현, 관리자 페이지 및 관련 api 요청에 제한을 두었습니다.
  • 별도의 CDN 서버를 두어 이미지 로딩시간 및 웹서버의 부하를 줄였습니다.
  • 계층형 트리구조를 활용해 댓글-대댓글 로직을 구현하였습니다.

    "포트폴리오 웹페이지"

  • vanila JS, CSS만을 이용한 싱글페이지 애플리케이션입니다.
  • Three.js를 이용해 웹 환경에서 3D 모델을 출력하고 브라우저 이벤트에 모델의 애니메이션을 연계하였습니다.
  • ease-in, ease-out 등 transition을 수행할 때 bezier-curve를 적용하여 사용자에게 보다 자연스럽게 보이도록 하였습니다.
  • 이미지 출력을 위한 별도의 CDN 서버를 두어 이미지 로딩시간 및 웹서버의 부하를 줄였습니다.

JS 기반의 다양한 프로젝트를 진행하였습니다.
